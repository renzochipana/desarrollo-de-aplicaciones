Conceptos

Las etiquetas semánticas en HTML son elementos que describen el propósito y el significado del contenido que contienen.

Contenido

Las etiquetas semánticas estructurales son esenciales para definir la estructura de un documento HTML.

Otros

Las etiquetas no semánticas en HTML son aquellas que no proporcionan información sobre el contenido que contienen.
